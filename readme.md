# BBS algorithm

## main.py 

Main script used to execute the algorithm or tests.

To execute BBS algorithm use:

```$ python3 main.py bbs [bit]```

To execute all tests use:

```$ python3 main.py bbs-tests [file]```

## bbs.py

Pyton File which contains primary number checking function, bbs algorithm function and tests functions.

## gen_serial.sh

Bash script used to generate file with serial of 20000 random bits using BBS algorithm

Outputs the file which could be used for bbs-tests
